# 必要なもの

* パソコン
* STiKA
* ステッカーシート
* アプリケーションシート
* ナイフ
* カッターマット

---
# 手順

1. STiKAを準備する
2. データをSTiKAに送りカットする
3. カットされたシートから余分な部分を取り除く
4. アプリケーションシートを使って貼付ける
5. 完成!!
